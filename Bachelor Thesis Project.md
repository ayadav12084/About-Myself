# Analysis of Distributed Deadlock Detection System 🛡️
**Academic Supervisor:** Joaquin Aguado (Constructor University)

**Grade:** 1.3 (90%) 

### Project Overview
This research-oriented project involved the formal modeling and verification of distributed deadlock detection algorithms to ensure system liveness and safety properties.

### Technical Implementation
***Modeling**: Developed models for different deadlock detection algorithms:- Centralized WFG(Wait-For-Graph), CMH (Chandy-Misra-Haas) and Hierachical Deadlock detection algorithm.

***Verification**: Conducted analysis of these models using the **SPIN Model Checker** and compared CMH and hierarchical deadlock detection algorithm to Centralied WFG deadlock detection algorithm.

***Analysis**: Performed rigorous testing on scalability, fault-tolerance, and error-recovery protocols.

### Key Outcomes
* Successfully compared these deadlock detection algorithm.
  
* Provided a verified framework for robust system recovery.
